# Guilty Pleasure Treats – Website

Static marketing site for the bakery. Deployed with the main project on Vercel.

- **Live:** The site is served at the project root (e.g. `https://guilty-pleasure-treats.vercel.app/`) when deployed.
- **Contents:** `index.html` (single-page), `styles.css`. Update the “App Store (when live)” link in `index.html` when the app is on the App Store.
- **API:** Backend stays under `/api/*` (products, orders, upload, health).
